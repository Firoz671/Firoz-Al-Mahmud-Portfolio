# Firoz Al Mahmud - Personal Portfolio

A modern, highly responsive, and meticulously designed personal portfolio website built to showcase my technical skills, projects, and professional background.

## 🚀 Live Demo
*(Not Yet Deployed)*

## 🔮 Features
- **Modern UI/UX**: Glassmorphism design, neon accents, and dark mode by default.
- **Dynamic Projects**: Custom React Hook integration with GitHub API to automatically fetch and display recent repositories.
- **Advanced Animations**: Fluid scrolling, text typing effects, and element reveals using Framer Motion.
- **Fully Responsive**: Flawless experience across Mobile, Tablet, and Desktop screens.

## 🛠️ Tech Stack
- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React (Icons)
- React Simple Typewriter

## 📦 Getting Started

### Prerequisites
Make sure you have Node.js (v16+) and npm installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Firoz671/Firoz-Al-Mahmud-Portfolio.git](https://github.com/Firoz671/Firoz-Al-Mahmud-Portfolio.git)
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
